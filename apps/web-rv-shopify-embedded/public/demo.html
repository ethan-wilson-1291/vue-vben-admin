Shopify RV App - Web Component Demo

This page demonstrates how to use the Shopify RV App as a Web Component.

Web Component Usage
Simply include the script and use the <shopify-rv-app> tag:
Integration Instructions

1. Include the script in your HTML:

<script src="./shopify-rv-app.umd.cjs"></script>

2. Use the custom element anywhere in your HTML:

<shopify-rv-app></shopify-rv-app>

3. For Shopify themes, add to your liquid templates:

{% comment %} In your theme's liquid file {% endcomment %}
<script src="{{ 'shopify-rv-app.umd.cjs' | asset_url }}"></script>
<shopify-rv-app></shopify-rv-app>
